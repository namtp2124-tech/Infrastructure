GPU serving & Inference (Day 16)

Tối ưu hóa tài nguyên GPU để phục vụ LLM với hiệu suất cao nhất:

  • vLLM (PagedAttention) loại bỏ phân mảnh bộ nhớ KV Cache.
  • SGLang (RadixAttention) tối ưu việc lưu nhớ ngữ cảnh lặp lại.
  • Áp dụng vGPU và MIG để phân bổ GPU hiệu quả cho các bài toán.
📂

Lakehouse & Vector Stores (Day 17-19)

Lưu trữ dữ liệu lớn tích hợp tìm kiếm ngữ nghĩa:

  • Apache Iceberg với REST Catalog (Apache Polaris) quản lý phân quyền.
  • Delta Lake với Deletion Vectors giảm thiểu số lần ghi lại file.
  • Qdrant/Milvus hỗ trợ index HNSW & IVF-PQ phục vụ tìm kiếm Vector cực nhanh.

8 Lỗi Hệ Thống Phổ Biến (Day 24)

Các điểm yếu mà AI Engine thường gặp phải:

  • Hallucination (Ảo tưởng), Bias (Thiên lệch), Sycophancy (Nịnh nọt).

  • Thiết lập Test set (Out-of-scope, Adversarial) bằng DeepEval/Ragas trước khi build.

Hệ Thống Phòng Thủ Đa Lớp (Day 25)

  • Input Layer: LlamaGuard 3 sàng lọc câu hỏi đầu vào độc hại.
  • Architectural Gates: Bật NeMo Guardrails (Colang) ngăn luồng hội thoại sai lệch.
  • UI/UX Warning: Hiển thị cảnh báo trực quan, cơ chế Undo hành động.

Glossary

Technical Glossary

08 / 08
RadixAttention
Kỹ thuật trong SGLang lưu trữ KV cache dưới dạng cây tiền tố (Prefix Tree), tái sử dụng cache cho system prompt và các file RAG lặp lại.
PagedAttention
Thuật toán phân trang bộ nhớ KV Cache LLM của vLLM, tương tự cơ chế phân trang RAM trong OS, triệt tiêu phân mảnh bộ nhớ.
Data Contracts
Cam kết ràng buộc cấu trúc dữ liệu (SLA, Schema) giữa bên cung cấp dữ liệu thô và bên huấn luyện/RAG, bảo vệ pipeline tránh bị sập bất ngờ.
Deletion Vectors
Tệp tin đánh dấu các dòng dữ liệu bị xóa trong định dạng Delta Lake/Iceberg, giúp truy vấn bỏ qua dòng đó mà không cần ghi lại file Parquet lớn.
Apache Polaris
Catalog siêu dữ liệu mã nguồn mở cho Lakehouse thực thi chuẩn Iceberg REST Catalog, đồng bộ hóa phân quyền truy cập.
GraphRAG
Mô hình truy xuất RAG kết hợp cơ sở tri thức đồ thị (Knowledge Graph), kết nối các thực thể để trả lời tốt câu hỏi liên kết đa điểm.
NeMo Guardrails
Thư viện của NVIDIA sử dụng ngôn ngữ Colang để lập trình luồng ranh giới an toàn cho chatbot, chống bẻ lái prompt (Jailbreak).
DeepEval
Khung kiểm thử đơn vị (Unit testing) mã nguồn mở tích hợp CI/CD để chấm điểm chất lượng prompt, hallucination, và bias của LLM.
Langfuse
Nền tảng mã nguồn mở quản lý và tracing luồng đi của các Agent, thay thế cho các giải pháp cloud độc quyền đắt đỏ.
OpenTelemetry GenAI (OTel)
Chuẩn semantic conventions của CNCF để xuất và theo dõi traces, spans, chi phí token của LLM một cách thống nhất.
